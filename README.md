# lemon-form


1. [在线预览](https://bojue.github.io/lemon-form)
2. [Github](https://github.com/bojue/lemon-form)

## 开发

```shell
git clone https://github.com/bojue/lemon-form.git
cd lemon-form
yarn  // or npm i 
npm run dev
```

## 启动

打开浏览器：http://localhost:3000/
